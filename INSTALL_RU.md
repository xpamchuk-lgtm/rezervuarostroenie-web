# Установка обновлённого проекта rezervuarostroenie.ru

## Вариант 1. Полный запуск через Docker Compose
1. Распакуйте архив на сервер.
2. Перейдите в каталог проекта.
3. Запустите:
   ```bash
   docker compose up -d --build
   ```
4. Проверьте:
   - главную страницу `/`
   - калькулятор `/calc/rvs`
   - базу знаний `/baza-znaniy/`
   - несуществующий путь должен отдавать `404`

## Вариант 2. Обновление только фронтенда
