# Расчет толщины стенки РВС: почему нельзя назначать одну толщину «на весь резервуар»

Стенка РВС работает не одинаково по высоте.

Внизу давление от продукта максимальное. Выше - меньше. Поэтому стенку считают по поясам, а не одной общей цифрой.

Базовая логика такая:

p_i = ρ · g · h_i

σ_θ = p_i · R / t_eff

t_eff = t_nom - c - Δt_minus

Условие:

σ_θ ≤ [σ] · φ

То есть номинальная толщина листа должна перекрывать не только расчет по прочности, но и:

- припуск на коррозию;
- минусовой допуск проката;
- минимальные требования стандарта;
- устойчивость стенки;
- технологию изготовления и монтажа.

Простой пример:

Если по нижнему поясу расчет дает 7,76 мм, коррозионный припуск 1,0 мм, а минусовой допуск 0,3 мм, то требуемая толщина уже:

t_req = 7,76 + 1,0 + 0,3 = 9,06 мм

Значит лист 8 мм не проходит, а ближайшее разумное решение - 10 мм, если остальные проверки также выполнены.

Главная ошибка - считать стенку только по гидростатике и забывать устойчивость, коррозию, допуски и местные зоны у патрубков.
