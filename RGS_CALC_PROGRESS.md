# Прогресс по калькулятору РГС

Дата фиксации: 2026-05-07

## Что уже изменено

- Калькулятор РГС перестроен ближе к логике РВС: центральная область показывает один выбранный раздел, а не всю форму сразу.
- Боковое меню разделов приведено к структуре:
  - Тип исполнения
  - Основные данные
  - Основные характеристики
  - Комплектация
  - Результаты
- Выбор типа исполнения РГСН / РГСП / РГСНД / РГСПД перенесен в отдельный раздел "Тип исполнения".
- В "Основные данные" добавлен выбор типоразмера / геометрического объема.
- Добавлено расчетное поле "Геометрический объем".
- При выборе типоразмера автоматически меняются:
  - диаметр;
  - общая длина;
  - количество колец жесткости / диафрагм.
- Если вручную меняется общая длина и включен режим авто-рекомендации, количество колец жесткости / диафрагм пересчитывается.
- Продукт выбирается из выпадающего списка, плотность подставляется автоматически.
- Добавлены регион и населенный пункт установки.
- Основные геометрические размеры в интерфейсе переведены в миллиметры.
- Перед отправкой в API размеры автоматически конвертируются из миллиметров в метры.
- Толщины, вылеты, днища, среда и режим работы собраны в "Основные характеристики".
- Патрубки, люки, лестница, площадка, изоляция, опоры, грунт, всплытие, хомуты и двустенное исполнение собраны в "Комплектацию".
- Результаты расчета вынесены в отдельный раздел.
- Бэкенд API для локальной проверки поднят на Python 3.12, так как Python 3.14 не смог установить закрепленный `pydantic-core` без Visual C++ linker.

## Проверки

- `tsc --noEmit` проходил без ошибок после изменений.
- `vite build` проходил успешно после изменений.
- Фронтенд отвечал на `http://127.0.0.1:5173/calc/rgs`.
- Бэкенд отвечал на `http://127.0.0.1:8000/api/health`.

## Технические примечания

- В корне проекта нет `.git`, поэтому коммит не создан.
- `git` не найден в PATH текущей системы.
- В `frontend/node_modules` были linux-версии optional-зависимостей Rollup/esbuild. Для локальной Windows-сборки были добавлены Windows-пакеты:
  - `node_modules/@rollup/rollup-win32-x64-msvc`
  - `node_modules/@esbuild/win32-x64`
